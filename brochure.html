What is Chatastrophe?

Six tools.
One playground.

Chatastrophe is a local web application that puts every dimension of LLM behavior under your control. Compare models, test personas, probe reasoning, tune parameters — all from a single interface that doesn't apologize for being opinionated.

Whether you're a researcher running systematic experiments, a developer benchmarking model choices, or just someone who wants to watch two AI characters argue about philosophy — Chatastrophe was built for you.

Core Features

Every way to talk to an AI.

Six purpose-built tabs, each designed to reveal something different about how language models think — and how they compare.

Chat — With Full Control

A clean chat interface with nothing held back. Every sampling parameter is exposed and adjustable in real time. See how the same question produces radically different answers when you dial up temperature or constrain output length.

  • Persistent conversation history with full search
  • Run the same prompt multiple times to compare outputs in the same thread
  • Log probabilities visualization — see token-level model confidence
  • Assign any custom persona via a rich system prompt
  • Works with OpenAI, Anthropic, Groq, and local Ollama models
📊

Parameter Sweep

Run the same prompt across a range of parameter values automatically. Visualize how temperature, top-p, or penalties change output quality and creativity across the full spectrum.

  • Pick any parameter to sweep
  • Set a range and step size
  • Compare all results in a scrollable grid
🧠

Reasoning Models

Built specifically for extended-thinking models. Watch the chain-of-thought unfold in real time — both the visible reasoning trace and the final answer stream simultaneously.

  • Supports o1, o3, and Claude extended thinking
  • Control reasoning effort and budget tokens
  • Expandable thinking trace per response
⚔️

Arena

Send the same prompt to two different models simultaneously and watch them respond side by side. No waiting, no switching tabs — pure parallel comparison in real time.

  • Any two models, any two providers
  • Responses stream in parallel
  • Ideal for capability benchmarking
Multi-Persona Modes

Put AI characters in a room together.

Chatastrophe's most unique features: multi-agent conversations and competitive scoring. These aren't chatbots — they're personalities with opinions.

🪑

Roundtable

Assemble up to four AI personas and give them a topic to discuss. Each agent has its own identity, model, and system prompt. They take turns responding to each other in a rolling conversation you can steer and extend.

  • Mix models and providers across agents
  • Tone sliders: Mood and Seriousness shape how the whole conversation feels
  • Continue with follow-up topics after the initial discussion
  • Conversations are saved and resumable from the sidebar
  • Export to PDF or JSON for sharing and archiving
  • Smart speaker selection — avoids back-to-back repetition for natural flow
🏆

Throwdown

The AI game show. Pick four personas: one is the Judge, three are Contestants. Each round, all three answer the same prompt and the Judge decides who gave the best response. Points accumulate over multiple rounds until a champion is crowned.

  • Judge is fixed per session — for unbiased, consistent scoring
  • 300+ built-in questions across 10 categories, auto-queued at session start
  • Live streaming responses — watch all three contestants answer at once
  • Real-time scoreboard throughout the competition
  • Full round history with judge reasoning preserved
  • Supports same persona with different models as contestants

"Throw GPT-4o, Claude Sonnet, and Llama 70B into a Throwdown with the same character prompt — and a fourth persona as Judge. Watch who wins at improv comedy, philosophy, and hot takes. The results will surprise you."

Use Cases

Who's it for?

Chatastrophe was built for people who want more than a chatbox. Here's who it's already for.

01  ·  Researchers

Systematic Model Evaluation

Run controlled parameter sweeps, fix seeds for reproducibility, and compare outputs across providers with identical prompts. Get real data, not anecdotes.

02  ·  Developers

Model Selection & Benchmarking

Throw the Arena at your actual use-case prompts. See which model gives better outputs for your specific task before committing to an API contract.

03  ·  Educators

Teaching LLM Behavior

Show students exactly how temperature changes creativity, how top-p affects diversity, and how the same persona prompt produces wildly different characters across models.

04  ·  Writers & Creatives

Character Workshopping

Use Roundtable to generate dialogue between characters at different tones and moods. Use Throwdown to find which model brings a character to life most vividly.

05  ·  Prompt Engineers

System Prompt Testing

Craft a persona, tune its system prompt, then run it through Throwdown against competing approaches to see which formulation wins on real prompts under judge scrutiny.

06  ·  The Curious

Just Seeing What Happens

Set Nietzsche, Sherlock Holmes, and a snarky chatbot against each other on "What is the meaning of a sandwich?" — and let an AI version of Oprah decide the winner.
